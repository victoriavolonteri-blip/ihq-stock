🔗 Conectar con Google Sheets
Los datos se guardan en una planilla de Google compartida con la cuenta del hospital. La configuración se hace una sola vez — después cualquier usuario abre esta misma app y ve los datos en tiempo real.
1
Crear la planilla en Google Sheets
Entrar a sheets.google.com con la cuenta del hospital y crear una planilla nueva. Puede llamarse IHQ Stock Anticuerpos.
2
Agregar el script de backend
En la planilla: Extensiones → Apps Script. Borrar todo el código y pegar el contenido del archivo IHQ_Stock_Script.gs. Guardar con Ctrl+S.
3
Publicar el script como Web App
En Apps Script: Implementar → Nueva implementación → Aplicación web.
• Ejecutar como: Yo (mi cuenta)
• Quién tiene acceso: Cualquier persona
Hacer clic en Implementar, autorizar los permisos y copiar la URL.
4
API Key de Anthropic (opcional — para leer remitos con IA)
Creá una cuenta en console.anthropic.com, andá a API Keys → Create Key y pegá la clave acá. Solo es necesaria para la función de leer remitos con la cámara.
💡 Para los demás usuarios del hospital: Compartiles este archivo HTML. Cada uno pega la misma URL del script en su primera apertura — queda guardada en el navegador y no necesitan repetirlo.
